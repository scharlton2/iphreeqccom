Using the IPhreeqc Object

The following example creates an IPhreeqc object, loads the phreeqc.dat  database and runs the ex1  example. 

  JScript

var IPhreeqc = new ActiveXObject("IPhreeqcCOM.Object");
IPhreeqc.LoadDatabase("phreeqc.dat");
IPhreeqc.OutputFileOn = true;
IPhreeqc.RunFile("ex1");


  VBScript

Dim IPhreeqc
Set IPhreeqc = CreateObject("IPhreeqcCOM.Object")
IPhreeqc.LoadDatabase("phreeqc.dat")
IPhreeqc.OutputFileOn = True
IPhreeqc.RunFile("ex1")


  Visual FoxPro

Dim IPhreeqc
IPhreeqc = CREATEOBJECT("IPhreeqcCOM.Object")
IPhreeqc.LoadDatabase("phreeqc.dat")
IPhreeqc.OutputFileOn = .T.
IPhreeqc.RunFile("ex1")

Remarks

Unlike the batch version of phreeqc, the output will be created in the file phreeqc.out. If the OutputFileOn property were not set to True no output files would have been created.
