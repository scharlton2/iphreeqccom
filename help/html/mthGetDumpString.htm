GetDumpString Method

Retrieves the string buffer containing DUMP output.

expression.GetComponentList()

expression    Required. An expression that returns an IPhreeqc object.

Remarks

The GetDumpString requires the DumpStringOn property set to True before calling a Run method in order to return a dump string.

Example

This example creates a solution and displays the resulting dump of the solution.


  JScript

var IPhreeqc = new ActiveXObject("IPhreeqcCOM.Object");
IPhreeqc.LoadDatabase("phreeqc.dat");
IPhreeqc.DumpStringOn = true;

var input = new String("");
input += "SOLUTION 1 Pure water     \n";
input += "EQUILIBRIUM_PHASES 1      \n";
input += "    Calcite 0 10          \n";
input += "SAVE solution 1           \n";
input += "SAVE equilibrium_phases 1 \n";
input += "DUMP                      \n";
input += "    -solution 1           \n";
input += "    -equilibrium_phases  1\n";

IPhreeqc.RunString(input);
var dump = IPhreeqc.GetDumpString();
WScript.Echo(dump);


  VBScript

Dim IPhreeqc, input, dump
Set IPhreeqc = CreateObject("IPhreeqcCOM.Object")
IPhreeqc.LoadDatabase("phreeqc.dat")
IPhreeqc.DumpStringOn = True

input = input & "SOLUTION 1 Pure water     " & vbNewLine
input = input & "EQUILIBRIUM_PHASES 1      " & vbNewLine
input = input & "    Calcite 0 10          " & vbNewLine
input = input & "SAVE solution 1           " & vbNewLine
input = input & "SAVE equilibrium_phases 1 " & vbNewLine
input = input & "DUMP                      " & vbNewLine
input = input & "    -solution            1" & vbNewLine
input = input & "    -equilibrium_phases  1" & vbNewLine

IPhreeqc.RunString(input)
dump = IPhreeqc.GetDumpString()
WScript.Echo(dump)
