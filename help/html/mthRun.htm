Run Method

Runs phreeqc using input that's been collected using the AccumulateLine method. Returns the number of errors encountered during the run.

expression.Run()

expression    Required. An expression that returns a phreeqc object.

Remarks

The Run method cannot be called until a database has been successfully loaded by one of the following methods: LoadDatabase or LoadDatabaseString.

Example

This example accumulates input and runs it through the phreeqc simulator.

Set Phreeqc = CreateObject("PhreeqcCOM.Phreeqc")
Phreeqc.LoadDatabase("phreeqc.dat")
Phreeqc.AccumulateLine("TITLE Example 2.--Temperature dependence of solubility")
Phreeqc.AccumulateLine("                  of gypsum and anhydrite")
Phreeqc.AccumulateLine("SOLUTION 1 Pure water")
Phreeqc.AccumulateLine("        pH      7.0")
Phreeqc.AccumulateLine("        temp    25.0")
Phreeqc.AccumulateLine("EQUILIBRIUM_PHASES 1")
Phreeqc.AccumulateLine("        Gypsum          0.0     1.0")
Phreeqc.AccumulateLine("        Anhydrite       0.0     1.0")
Phreeqc.AccumulateLine("REACTION_TEMPERATURE 1")
Phreeqc.AccumulateLine("        25.0 75.0 in 51 steps")
Phreeqc.AccumulateLine("SELECTED_OUTPUT")
Phreeqc.AccumulateLine("        -file   ex2.sel")
Phreeqc.AccumulateLine("        -temperature")
Phreeqc.AccumulateLine("        -si     anhydrite  gypsum")
Phreeqc.AccumulateLine("END")
Phreeqc.Run
