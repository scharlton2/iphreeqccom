LogStringOn Property

True to store data normally sent to the log file to a string buffer for retrieval. The initial setting is False. Read/write Boolean.

Remarks

The LogStringOn property is used only by the Run methods: RunAccumulated, RunFile and RunString.

Example

This example equilibrates pure water with calcite and displays the log string.


  JScript

var iphreeqc = new ActiveXObject("IPhreeqcCOM.Object");
iphreeqc.LoadDatabase("phreeqc.dat");
iphreeqc.LogStringOn = true;

var input = new String("");
input += "SOLUTION 1 Pure water     \n";
input += "EQUILIBRIUM_PHASES 1      \n";
input += "    Calcite 0 10          \n";
input += "KNOBS                     \n";
input += "    -logfile TRUE         \n";

if (iphreeqc.RunString(input) == 0) {
    WScript.Echo(iphreeqc.GetLogString());
}

  VBScript

Dim iphreeqc, sInput

Set iphreeqc = CreateObject("IPhreeqcCOM.Object")
iphreeqc.LoadDatabase("phreeqc.dat")
iphreeqc.LogStringOn = True

sInput = ""
sInput = sInput & "SOLUTION 1 Pure water     " & vbNewLine
sInput = sInput & "EQUILIBRIUM_PHASES 1      " & vbNewLine
sInput = sInput & "    Calcite 0 10          " & vbNewLine
sInput = sInput & "KNOBS                     " & vbNewLine
sInput = sInput & "    -logfile TRUE         " & vbNewLine

If (iphreeqc.RunString(sInput) = 0) Then
    WScript.Echo(iphreeqc.GetLogString())
End If
