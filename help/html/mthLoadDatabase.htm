LoadDatabase Method

Loads a phreeqc database into the phreeqc object. Returns the number of errors encountered while loading the database.

expression.LoadDatabase(FileName )

expression    Required. An expression that returns a phreeqc object.

FileName   Required String. The file name of the phreeqc database to load.

Example

This example loads the "phreeqc.dat" database into the phreeqc object.

Set Phreeqc = CreateObject("PhreeqcCOM.Phreeqc")
Phreeqc.LoadDatabase("phreeqc.dat")
