SelectedOutputOn Property

True to turn on the SELECTED_OUTPUT file. The file will be named "selected.out" unless it was specified within the SELECTED_OUTPUT keyword block. The default value is False. Read/write Boolean.

Remarks

The SelectedOutputOn property is used only by the Run methods: Run, RunFile and RunString.

Example

This example runs "ex2" with the selected_output file turned on.

set Phreeqc = CreateObject("PhreeqcCOM.Phreeqc")
Phreeqc.LoadDatabase("phreeqc.dat")
Phreeqc.SelectedOutputOn = True
Phreeqc.RunFile("ex2")
