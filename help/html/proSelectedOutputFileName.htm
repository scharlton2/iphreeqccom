SelectedOutputFileName Property

Returns or sets the name of the selected-output file. The initial setting is selected.id.out, where id is obtained from Id. Read/write String.

Remarks

The SelectedOutputFileOn property must be set to True before calling a Run method.

Example

This example runs ex2  with the selected-output file turned on.


  JScript

var iphreeqc = new ActiveXObject("IPhreeqcCOM.Object");
iphreeqc.LoadDatabase("phreeqc.dat");
iphreeqc.SelectedOutputFileOn = true;
if (iphreeqc.RunFile("ex2") == 0) {
    WScript.Echo("See " + iphreeqc.SelectedOutputFileName + ".");
}


  VBScript (Deprecated)

Dim iphreeqc
Set iphreeqc = CreateObject("IPhreeqcCOM.Object")
iphreeqc.LoadDatabase("phreeqc.dat")
iphreeqc.SelectedOutputFileOn = True
If (iphreeqc.RunFile("ex2") = 0) Then
    WScript.Echo("See " & iphreeqc.SelectedOutputFileName & ".")
End If
