SelectedOutputStringOn Property

True to store data normally sent to the selected-output file to a string buffer for retrieval. The initial setting is False. Read/write Boolean.

Remarks

The SelectedOutputStringOn property is used only by the Run methods: RunAccumulated, RunFile and RunString.

Example

This example runs ex2  and displays the selected-output string.


  JScript

var iphreeqc = new ActiveXObject("IPhreeqcCOM.Object");
iphreeqc.LoadDatabase("phreeqc.dat");
iphreeqc.SelectedOutputStringOn = true;
if (iphreeqc.RunFile("ex2") == 0) {
    WScript.Echo(iphreeqc.GetSelectedOutputString());
}


  VBScript

Dim iphreeqc
Set iphreeqc = CreateObject("IPhreeqcCOM.Object")
iphreeqc.LoadDatabase("phreeqc.dat")
iphreeqc.SelectedOutputStringOn = True
If (iphreeqc.RunFile("ex2") = 0) Then
    WScript.Echo(iphreeqc.GetSelectedOutputString())
End If
