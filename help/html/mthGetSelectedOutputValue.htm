GetSelectedOutputValue Method

Loads a phreeqc database into the phreeqc object. Returns the number of errors encountered while loading the database.

expression.GetSelectedOutputValue(Row, Column)

expression    Required. An expression that returns a phreeqc object.

Row   Required Long. Zero-based row index.

Column   Required Long. Zero-based column index.

Example

This example runs the "ex2" input file and writes the selected output to stdout.


  JScript

var Phreeqc = new ActiveXObject("PhreeqcCOM.Phreeqc");
Phreeqc.LoadDatabase("phreeqc.dat");
Phreeqc.RunFile("ex2");

for (i = 0; i < Phreeqc.RowCount; ++i)
{
    for (j = 0; j < Phreeqc.ColumnCount; ++j)
    {
        WScript.Stdout.Write(Phreeqc.GetSelectedOutputValue(i, j));
        WScript.Stdout.Write(" ");
    }
    WScript.Stdout.WriteLine();
}


  VBScript

Dim Phreeqc, i, j
Set Phreeqc = CreateObject("PhreeqcCOM.Phreeqc")
Phreeqc.LoadDatabase("phreeqc.dat")
Phreeqc.RunFile("ex2")

For i = 0 To Phreeqc.RowCount - 1
    For j = 0 To Phreeqc.ColumnCount - 1
        If (IsObject(WScript.Stdout)) Then
            WScript.Stdout.Write(Phreeqc.GetSelectedOutputValue(i, j) & " ")
        End If
    Next
    If (IsObject(WScript.Stdout)) Then
        WScript.Stdout.WriteLine
    End If
Next
