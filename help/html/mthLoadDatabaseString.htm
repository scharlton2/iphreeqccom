LoadDatabaseString Method

Loads a phreeqc database into the phreeqc object. Returns the number of errors encountered while loading the database.

expression.LoadDatabaseString(Input )

expression    Required. An expression that returns a phreeqc object.

Input   Required. The string containing phreeqc database.

Example

This example reads the "phreeqc.dat" database into a String and loads it into the phreeqc object.

Const ForReading = 1

Set fso = CreateObject("Scripting.FileSystemObject")
Set ts = fso.OpenTextFile("phreeqc.dat", ForReading)
strDB = ts.ReadAll()

Set Phreeqc = CreateObject("PhreeqcCOM.Phreeqc")
Phreeqc.LoadDatabaseString(strDB)
