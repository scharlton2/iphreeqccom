RunFile Method

Runs phreeqc using the given file as input. Returns the number of errors encountered during the run.

expression.RunFile(FileName )

expression    Required. An expression that returns a phreeqc object.

FileName   Required. The name of the phreeqc input file to run.

Remarks

The RunFile method cannot be called until a database has been successfully loaded by one of the following methods: LoadDatabase or LoadDatabaseString.

Example

This example runs the "ex1" input file in the phreeqc simulator.

Set Phreeqc = CreateObject("PhreeqcCOM.Phreeqc")
Phreeqc.LoadDatabase("phreeqc.dat")
Phreeqc.OutputOn = True
Phreeqc.RunFile("ex1")
