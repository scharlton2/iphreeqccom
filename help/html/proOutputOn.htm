OutputOn Property

True to turn on output to the "phreeqc.out" file . The default value is False. Read/write Boolean.

Remarks

The OutputOn property is used only by the Run methods: Run, RunFile and RunString.

Example

This example runs "ex2" with the "phreeqc.out" file turned on.


  JScript

var Phreeqc = new ActiveXObject("PhreeqcCOM.Phreeqc");
Phreeqc.LoadDatabase("phreeqc.dat");
Phreeqc.OutputOn = true;
Phreeqc.RunFile("ex2");


  VBScript

Dim Phreeqc
Set Phreeqc = CreateObject("PhreeqcCOM.Phreeqc")
Phreeqc.LoadDatabase("phreeqc.dat")
Phreeqc.OutputOn = True
Phreeqc.RunFile("ex2")


  Visual FoxPro

Phreeqc = CREATEOBJECT("PhreeqcCOM.Phreeqc")
Phreeqc.LoadDatabase("phreeqc.dat")
Phreeqc.OutputOn = .T.
Phreeqc.RunFile("ex2")
