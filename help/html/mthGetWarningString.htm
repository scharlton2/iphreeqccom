GetWarningString Method

Returns a string containing any possible warning messages that occured during the last invokation of the following methods: LoadDatabase, LoadDatabaseString, RunAccumulated, RunFile, RunString.

expression.GetWarningString

expression    Required. An expression that returns an IPhreeqc object.

Example

The example loads the phreeqc.dat database and attempts to use the DATABASE keyword to set the database to wateq4f.dat. A warning is displayed stating that the DATABASE keyword is ignored by IPhreeqc.


  JScript

var IPhreeqc = new ActiveXObject("IPhreeqcCOM.Object");
IPhreeqc.LoadDatabase("phreeqc.dat");
IPhreeqc.AccumulateLine("DATABASE wateq4f.dat");
IPhreeqc.AccumulateLine("SOLUTION 1");
IPhreeqc.RunAccumulated();
if (IPhreeqc.GetWarningString().length != 0)
{
    WScript.Echo(IPhreeqc.GetWarningString());
}


  VBScript

Dim IPhreeqc
Set IPhreeqc = CreateObject("IPhreeqcCOM.Object")
IPhreeqc.LoadDatabase("phreeqc.dat")
IPhreeqc.AccumulateLine("DATABASE wateq4f.dat")
IPhreeqc.AccumulateLine("SOLUTION 1")
IPhreeqc.RunAccumulated()
If (Len(IPhreeqc.GetWarningString()) <> 0) Then
    WScript.Echo(IPhreeqc.GetWarningString())
End If
