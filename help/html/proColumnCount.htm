ColumnCount Property

Returns the number of columns in the selected output-array. Read-only Long.

Remarks

Accessing selected outputs having a user number other than 1 requires setting the CurrentSelectOutputUserNumber property appropriately.

Example

This example runs ex2  and displays the number of columns in the selected-output array.


  Powershell

[System.IO.Directory]::SetCurrentDirectory($PSScriptRoot)
$iphreeqc = New-Object -ComObject IPhreeqcCOM.Object
$iphreeqc.LoadDatabase("phreeqc.dat") | Out-Null
$iphreeqc.RunFile("ex2") | Out-Null
Write-Host(-join("The current selected-output array contains ", `
    "$($iphreeqc.ColumnCount) columns."))

  JScript

var iphreeqc = new ActiveXObject("IPhreeqcCOM.Object");
iphreeqc.LoadDatabase("phreeqc.dat");
iphreeqc.RunFile("ex2");
WScript.Echo("The current selected-output array contains " +
    iphreeqc.ColumnCount + " columns.");


  VBScript (Deprecated)

Dim iphreeqc
Set iphreeqc = CreateObject("IPhreeqcCOM.Object")
iphreeqc.LoadDatabase("phreeqc.dat")
iphreeqc.RunFile("ex2")
WScript.Echo("The current selected-output array contains " & _
    iphreeqc.ColumnCount & " columns.")
