ColumnCount Property

Returns the number of columns in the selected output array. Read-only Long.

Example

This example runs "ex2" and displays the number of columns contained within the selected output array.

sPath = "C:\Program Files\USGS\phreeqc-2.12.1\"
set Phreeqc = CreateObject("PhreeqcCOM.Phreeqc")
Phreeqc.LoadDatabase sPath & "phreeqc.dat"
Phreeqc.RunFile sPath & "examples\ex2"
MsgBox "The current selected output array contains " & _
    Phreeqc.ColumnCount & " columns."
