ErrorFileName Property

Returns or sets the name of the error file. The initial setting is phreeqc.id.err, where id is obtained from Id. Read/write String.

Remarks

The ErrorFileOn property must be set to True before calling a Run method.

Example

The example attempts to run the ex1  input file, fails, and displays the name of the error file.


  JScript

try {
    var iphreeqc = new ActiveXObject("IPhreeqcCOM.Object");
    iphreeqc.ErrorFileOn = true;
    iphreeqc.RunFile("ex1");
}
catch(e) {
    WScript.Echo("See " + iphreeqc.ErrorFileName + ".");
}

  VBScript

Dim iphreeqc
On Error Resume Next
Set iphreeqc = CreateObject("IPhreeqcCOM.Object")
iphreeqc.ErrorFileOn = True
If (iphreeqc.RunFile("ex1") <> 0) Then
    WScript.Echo(("See " & iphreeqc.ErrorFileName & ".")
End If
