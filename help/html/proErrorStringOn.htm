ErrorStringOn Property

True to store data normally sent to the error file to a string buffer for retrieval. The initial setting is True. Read/write Boolean.

Remarks

The ErrorStringOn property is used only by the Run methods: RunAccumulated, RunFile and RunString.

Example

The example attempts to run the ex1  input file, fails, and displays the error string.


  Powershell

xxxPASTE_HERExxx


  JScript

try {
    var iphreeqc = new ActiveXObject("IPhreeqcCOM.Object");
    iphreeqc.ErrorStringOn = true;
    iphreeqc.RunFile("ex1");
}
catch (e) {
    WScript.Echo(iphreeqc.GetErrorString());
}

  VBScript (Deprecated)

Dim iphreeqc
