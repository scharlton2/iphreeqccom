GetSelectedOutputString Method

Retrieves the string buffer containing selected-output.

expression.GetSelectedOutputString()

expression    Required. An expression that returns an IPhreeqcCOM object.

Remarks

The GetSelectedOutputString method requires the SelectedOutputStringOn property be set to True before calling one of the Run methods.

Accessing selected outputs having a user number other than 1 requires setting the CurrentSelectOutputUserNumber property appropriately.

Example

This example runs the ex2  input file and displays the selected-output string.


  Powershell

[System.IO.Directory]::SetCurrentDirectory("$(Get-Location)")
$iphreeqc = New-Object -ComObject IPhreeqcCOM.Object

$iphreeqc.LoadDatabase("phreeqc.dat") | Out-Null
$iphreeqc.SelectedOutputStringOn = $true

if ($iphreeqc.RunFile("ex2") -eq 0) {
    Write-Host $iphreeqc.GetSelectedOutputString()
}

  JScript

var iphreeqc = new ActiveXObject("IPhreeqcCOM.Object");
iphreeqc.LoadDatabase("phreeqc.dat");
iphreeqc.SelectedOutputStringOn = true;
iphreeqc.RunFile("ex2");

if (iphreeqc.RunFile("ex2") == 0) {
    WScript.Echo(iphreeqc.GetSelectedOutputString());
}

  VBScript (Deprecated)

Dim iphreeqc

Set iphreeqc = CreateObject("IPhreeqcCOM.Object")
iphreeqc.LoadDatabase("phreeqc.dat")
iphreeqc.SelectedOutputStringOn = True

If (iphreeqc.RunFile("ex2") = 0) Then
    WScript.Echo(iphreeqc.GetSelectedOutputString())
End If
