Version Property

Returns the version number of the IPhreeqcCOM object. Read-only String.

Example

This example creates an IPhreeqcCOM object and displays the Version.


  Powershell

PASTE_HERE


  JScript

var iphreeqc = new ActiveXObject("IPhreeqcCOM.Object");
WScript.Echo(iphreeqc.Version);

  VBScript (Deprecated)

Dim iphreeqc
Set iphreeqc = CreateObject("IPhreeqcCOM.Object")
WScript.Echo iphreeqc.Version
