GetLastErrorString Method

Returns a string containing any possible error messages that occured during the last invokation of the following methods: AccumulateLine, GetSelectedOutputValue, LoadDatabase, LoadDatabaseString, Run, RunFile, RunString.

expression.GetLastErrorString

expression    Required. An expression that returns a phreeqc object.

Example

The example attempts to run the "ex1" input file, fails, and displays the reason for failure (no database is loaded).


  JScript

try
{
    var shell = new ActiveXObject("wscript.shell");
    var Phreeqc = new ActiveXObject("PhreeqcCOM.Phreeqc");
    Phreeqc.RunFile("ex1");
}
catch(e)
{
    shell.Popup(Phreeqc.GetLastErrorString());  // could also use e.message
}


  VBScript

Dim Phreeqc
On Error Resume Next
Set Phreeqc = CreateObject("PhreeqcCOM.Phreeqc")
If (Phreeqc.RunFile("ex1") <> 0) Then
    MsgBox(Phreeqc.GetLastErrorString)
End If
