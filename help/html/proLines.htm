Lines Property

Returns the accumlated text contained within the phreeqc objects input buffer. Read-only String.

Example

This example loads some keyword input into the phreeqc object and displays the result.

set Phreeqc = CreateObject("PhreeqcCOM.Phreeqc")
Phreeqc.AccumulateLine("SOLUTION 1")
Phreeqc.AccumulateLine("END")
MsgBox("The input to run is:" & Phreeqc.Lines)
