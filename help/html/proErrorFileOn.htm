ErrorFileOn Property

True to turn on output to the phreeqc error file ErrorFileName. The initial setting is False. Read/write Boolean.

Remarks

The ErrorFileOn property is used only by the Run methods: RunAccumulated, RunFile and RunString.

Example

The example attempts to run the ex1  input file, fails, and writes the error messages to the error file ErrorFileName (no database is loaded).


  Powershell

xxxPASTE_HERExxx


  JScript

try {
    var iphreeqc = new ActiveXObject("IPhreeqcCOM.Object");
    iphreeqc.ErrorFileOn = true;
    iphreeqc.RunFile("ex1");
}
catch(e) {
    WScript.Echo("See " & iphreeqc.ErrorFileName & ".");
}

  VBScript (Deprecated)

Dim iphreeqc
On Error Resume Next
Set iphreeqc = CreateObject("IPhreeqcCOM.Object")
iphreeqc.ErrorFileOn = True
If (iphreeqc.RunFile("ex1") <> 0) Then
    WScript.Echo("See " & iphreeqc.ErrorFileName & ".")
End If
