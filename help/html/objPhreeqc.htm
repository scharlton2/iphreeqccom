Using the Phreeqc Object

The following example creates a Phreeqc object, loads the "phreeqc.dat" database and runs the "ex1" example. 
Set Phreeqc = CreateObject("PhreeqcCOM.Phreeqc")
Phreeqc.LoadDatabase("phreeqc.dat")
Phreeqc.OutputOn = True
Phreeqc.RunFile("ex1")

Remarks

Unlike the batch version of phreeqc, the output will be created in the file "phreeqc.out". If the OutputOn property were not set to True no output files would have been created.
